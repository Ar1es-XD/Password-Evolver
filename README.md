# 🔐 Password Evolver

A terminal-based simulation that evolves random characters toward a target password — one lucky guess at a time.

Inspired by the concept of genetic algorithms, each character slot independently randomizes until it matches the target. Watch it converge in real time with a live color-coded display and performance stats.

---

## ✨ Features

- 🎯 **Character-by-character evolution** — locked chars stay, unmatched chars keep randomizing
- 🌈 **Color-coded display** — green for matched, red for unmatched
- 📊 **Live progress bar** — see how close you are at a glance
- ⚡ **Performance stats** — attempts, elapsed time, and tries/sec
- 🖥️ **Clean terminal UI** — redraws every 50 attempts, no scroll spam
